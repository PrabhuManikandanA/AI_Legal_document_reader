# AI-Powered Legal Document Review System

This project demonstrates a multi-agent AI system built using Azure AI services to review legal documents,
extract clauses, validate compliance using RAG, and generate grounded summaries.

## Tech Stack
- Azure AI Agent Service
- Azure AI Foundry
- Azure AI Search
- Azure Blob Storage
- Azure OpenAI
- Semantic Kernel
- Microsoft Agent Framework

## Agents
1. Document Ingestion Agent
2. Clause Extraction Agent
3. Compliance Validation Agent (RAG)
4. Summary Agent

## How to Run (Demo)
1. Upload legal documents to Blob Storage
2. Index documents in Azure AI Search
3. Trigger agent workflow from Azure AI Foundry
4. Review outputs and summaries

Screenshots and prompts are included for reference.
